# Electronic Lock RP2040

An electronic lock simulation built with **CircuitPython** on a **Raspberry Pi RP2040**. The project uses a 4×4 matrix keypad to enter a password, LEDs to provide visual feedback, and an active buzzer for audible notifications.

## Features

- Password entry using a 4×4 matrix keypad.
- Password validation.
- Visual feedback through 8 LEDs.
- Audible feedback using an active buzzer.
- Access granted and access denied indications.
- Delete the last entered digit.
- Written in CircuitPython.

## Hardware

- Raspberry Pi RP2040
- HC-35S 4×4 Matrix Keypad
- 3-Pin Active Buzzer
- 8 LEDs
- Resistors
- Jumper Wires

## Software

- CircuitPython
- Visual Studio Code
- Adafruit CircuitPython Libraries

## Library Dependencies

This project requires:

- `adafruit_matrixkeypad`

The remaining modules (`board`, `digitalio`, and `time`) are included with CircuitPython.

## Wiring

| Component | GPIO |
|----------|------|
| Matrix Keypad Columns | GP2, GP3, GP4, GP5 |
| Matrix Keypad Rows | GP6, GP7, GP8, GP9 |
| Confirm Button | GP10 |
| Delete Button | GP11 |
| Reserved | GP12, GP13 |
| Active Buzzer | GP15 |
| LEDs | GP0, GP1, GP16–GP21 |

## How to Run

1. Install CircuitPython on the RP2040.
2. Copy `code.py` to the `CIRCUITPY` drive.
3. Install the required library in the `lib` folder.
4. Connect the hardware according to the wiring table.
5. Reset the board.
